## [1.0.0] — 2026-06-16

### Added
- Initial public release of `/enhance` Claude Code slash command
- Full 5-phase prompt enhancement pipeline (Ecosystem Scan, Task Classification, Knowledge Generation, Tree of Thoughts, Quality Checklist)
- 10-step mandatory Chain-of-Thought reasoning before output
- 16 prompting techniques integrated from DAIR.AI and Google Cloud
- Comprehensive technique reference in `docs/techniques.md`
- Three worked examples (legal, code, research)
- Issue and pull request templates
- GitHub Actions CI workflow for markdown linting
- MIT licence
- Contributing guidelines and Code of Conduct
